# Flask Basic App

A simple web application built using **Python** and **Flask**.  
This project is intended for learning the basics of Flask, routing, and running a web server.

A basic Gym Management System website using Flask, HTML, and CSS. The website included:
- A Home page displaying a welcome message and gym highlights.
- An About Us page sharing details about the gym's mission, team, and facilities.
- A Membership page showcasing membership plans and services.


---

## Features

- Lightweight Flask framework
- Basic routing
- Easy to understand project structure
- Suitable for beginners

---

## Technologies Used

- Python 3.x  
- Flask  

---

## Starting the web app  by

```bash
python app.py
```
